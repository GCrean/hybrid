Hybrid

Hybird is an lightweight IM framework, currently supports China Mobile Fetion protocol and xmpp protocol.

Download this project as a .zip file Download this project as a tar.gz file

How to install

So far, you can only install hybrid from source code, fetch the code and compile in the following way:

$git clone git://github.com/levin108/hybrid.git

$cd hybrid
$mkdir -p build
$cd build
$cmake ..
$make
#make install
